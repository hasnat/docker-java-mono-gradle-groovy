
### hasnat/java-mono-gradle-groovy
Docker container containing java, mono, gradle and groovy

## Getting Started

To build docker
```
docker run -d \
    --name app-1 \
    -e HOME=/app \
    -v /app/source/path/on/host:/app/ \
    -v /config/gradle-init:/root/.gradle/ \
    hasnat/java-mono-gradle-groovy
```

To start or attach
```
docker exec -i -t ent-dotnet /bin/bash
```
